# Milestones

A milestone defines a runnable product increment. It records stable scope and acceptance criteria. Implementation tasks belong in issues or the project board.

Every milestone document must define:

- Outcome
- In scope
- Out of scope
- Deliverables
- Demonstration
- Completion criteria

The authoritative status is in [`roadmap.md`](../roadmap.md). A milestone cannot be marked `Complete` until every completion criterion and required gate has passed.
